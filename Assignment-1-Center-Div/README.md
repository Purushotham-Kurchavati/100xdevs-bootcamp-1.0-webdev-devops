# Assignment 1: Center a Div

This project demonstrates how to center a div both horizontally and vertically using modern CSS layout techniques. A parent container is created to occupy the full viewport height, and Flexbox is applied to align the child div precisely at the center. By using properties such as display: flex, justify-content: center, and align-items: center, the layout becomes simple, clean, and highly effective.

The centered div is styled with fixed dimensions, background color, and text alignment to clearly visualize the result. This approach ensures that the content remains perfectly centered regardless of screen size, making it a reliable and commonly used method in real-world web development.

Overall, this assignment highlights the importance of layout systems like Flexbox in building responsive and well-structured user interfaces.
